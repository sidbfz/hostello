# Hostello

Simple static files for the Hostello project — HTML, CSS and JS used for small demo pages and UI prototypes.

Files in this repository:
- `hostello-hero.html` — hero section demo
- `hostello-hero.css` — hero styles
- `hostello-hero.js` — hero behaviour
- various demo HTML files (`sticker.html`, `milo-profile.html`, etc.)

How to use
- Open the HTML files in a browser to preview.
